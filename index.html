⚠️ Bitcoin Core 30.0 Warning

Important: Bitcoin Core 30.0 increases the OP_RETURN size limit from 83 to 100,000 bytes, potentially enabling more data storage on Bitcoin and increasing network fees. This change may impact Bitcoin's primary use as a monetary network.

Recommendations:

  • Stay on Bitcoin Core 29.x or earlier versions
  • If upgrading to Core 30.0, add -datacarriersize=83 to your configuration to maintain the original limit
  • Consider running Bitcoin Knots as an alternative
